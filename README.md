# React Express Boilerplate

<p>Boilerplate for a typical setup for building an app using React and Express built with create-react-app and express generator. In addition to React, Express it uses knex as the query builder and Postgresql</p>

To start client:

- cd client && yarn
- yarn start

To start server:

- cd server && npm install
- npm start

Client runs on port 3000 and server runs on port 3001
